# itDict

## Overview
A dictionary project which is build with Kony Mobile Framework (Multiplatform Mobile Framework).

## Functions

This includes:

- You can translate word from english to japanese.
- You can put a word to a favourite using touch in grey star in left of a word.
- You can contribute your word with the input in the below each word. Ex: Depending on the context, words are also used differently.
- You can manage your favourite word in the dashboard.
- You can remove all history which is in the main screen.

# Techniques
### Kony Animations
This app demonstrate the animations which are supported by the Kony platform
In this app we have coverd the below animations:
- Move animation
- Scale Animation
- Transform 5.Dual Animation
### Kony segment
Segment Sample app with use case:
- Getting segment record details on button click using context parameter

**To Run This App in Kony Visualizer:**

1. Download the project zip file.
2. Unzip the project to any folder.
3. Launch Kony Visualizer
4. Import it into Kony Visualizer Starter Edition 7.3 by selecting itDict.zip or copy the folder to C:\Users\user-name\Kony Visualizer\konyworkspace on Windows.
5. Run the app.
6. Launch in Functional Preview through Local account or Cloud.

Local :

1. Enter the IP address.
2. Enter the port value.
3. Enter the Application name.

Cloud :

1. Publish App Preview.
2. Select the channels to publish.
3. Enter the generated code in visualizer App Preview in your Device.
